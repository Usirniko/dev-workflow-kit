# 🐧 AI企鹅养成 - 大学生成长伙伴

> 一个面向大学生的AI企鹅养成Web应用，通过与AI对话，让企鹅随你的学习与成长内容不断进化。

## 📖 产品概述

AI企鹅养成是一款创新型学习陪伴应用。大学生通过日常与AI聊天（学习讨论、职业规划、心情分享等），让虚拟企鹅伙伴根据对话内容逐步成长与进化。企鹅不仅是一个可爱的虚拟宠物，更是你大学生涯的见证者和成长伙伴。

**核心理念**：每一次有深度的对话，都是企鹅和你共同的成长。

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 🥚 领蛋 | 首次访问领取专属企鹅蛋 |
| 🎓 年级选择 | 大一至大四，四个成长阶段 |
| 📝 命名卡片 | 为企鹅取一个专属名字 |
| 💬 AI聊天 | 接入 DeepSeek API（需用户自行配置 API Key），智能对话 |
| 📚 离线问答库 | 51条预设问答（含 10 条思辨反问式问答），无需 API 也能聊天 |
| 🐧 企鹅进化 | 根据聊天内容动态变化外观（蛋→幼崽→青少年→成年） |
| 📊 特征系统 | 多维度评估成长状态（知识力/创造力/社交力/行动力/心态值） |
| ⚖️ 消息质量权重 | 长度×因果链×反思词 三因子乘积，奖励深度交流 |
| 🧠 思辨引导 | 四个年级梯度化思辨规则，企鹅从「告知者」变为「思考伙伴」 |
| 🏷️ 企鹅名字全链路动态化 | 用户取名后 AI/离线/System Prompt 全线同步使用 |
| 📖 聊天段落渲染 | 自动识别列表/正文，分段显示 + 智能首行缩进 |
| 📓 日记 | 记录每日学习与成长 |
| 📋 简历 | 腾讯校招简历模板，边聊边填 |
| 🎨 装扮 | 关键词驱动的企鹅配饰系统（14种配饰） |
| 🏅 活动成就 | 20个年级活动 + 5项每日任务 + 10个成就徽章 |
| 🕐 实时时钟 | 首页显示当前时间 HH:MM:SS |
| 🐧 企鹅问候 | 按时段（早/中/下/晚/夜）随机切换问候语 |
| 🕐 AI 时间感知 | AI 感知当前日期时间，深夜不说"上课"，周末不聊考试 |
| 🎬 动画控制 | 帧跳过机制，企鹅动画速度优雅放缓 |
| 💾 本地存储 | localStorage持久化数据 |
| 🚀 大四校招指南 | 4个可视化板块（校招流程/面经汇总/Offer决策/焦虑疏导），卡片式排版 |

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`
2. 首次访问会进入领蛋页面
3. 选择你的年级后即可开始聊天
4. 所有数据自动保存在浏览器本地

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **UI规范**: TDesign Design Tokens
- **AI引擎**: DeepSeek API (deepseek-chat)，需用户在设置中自行配置 API Key
- **数据存储**: localStorage
- **动画**: CSS Animations + Canvas 2D

## 📋 浏览器兼容

- Chrome 90+
- Edge 90+
- Safari 14+
- Firefox 88+
- 移动端浏览器（iOS Safari / Android Chrome）
