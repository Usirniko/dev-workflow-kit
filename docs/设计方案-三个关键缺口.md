# 🎯 三个关键缺口 · 完整设计方案

> 创建日期：2026-06-13  
> 前置评估：[docs/开发日志.md](./开发日志.md) v3.3 → v4.4

---

## 背景回顾

当前应用在「全年级覆盖」「情感链接」方面已做得很出色，但以下三个缺口制约了产品从"有趣的信息工具"升级为"真正的雇主品牌漏斗"：

| # | 缺口 | 影响 |
|---|------|------|
| 1 | 大一~大三卡片深度不足 | 大四有 4 个 HTML 可视化板块，大一~大三相对单薄 |
| 2 | 卡片缺少进度与交互闭环 | 看完就走了，没有「我在成长」的正反馈 |
| 3 | 缺少真实连接桥梁 | 没有留资入口，线上陪伴无法转化为线下机会 |

---

## 缺口 1：大一~大三卡片内容升级

### 1.1 现状诊断

| 年级 | 板块数 | 可视化类型 | 评优 |
|------|:--:|------|:--:|
| 大一·好奇探索期 | 3 | tabs(4) / biz-grid(7) / checklist(10) | 行业全景图 4 个 tab 面内嵌动态渲染，属「半成品」——`html: ''` 的部分依赖 JS 函数渲染，展开后可能空 |
| 大二·方向探索期 | 3 | role-detail-cards(7) / major-career-map(8×3) / skill-tree(6×8) | 最扎实的年级，但技能树是「阅读型」清单，缺少交互深度 |
| 大三·实习准备期 | 4 | role-detail-cards(4) / html / role-detail-cards(5) / html | 简历写作指南和实习时间线是 HTML 直接内嵌，信息密度高但视觉单一 |
| **大四·求职冲刺期** | **4** | **全部 html（4 个完整 HTML+CSS 板块）** | **标杆，268 行 CSS** |

**结论**：大一是最需要加强的低年级入口，大三的 html 内嵌板块可升级为结构化 visualData。

## 缺口 2：卡片进度系统

### 2.1 设计目标

学生打开一个年级卡片 → 看到 5~6 个板块 → 展开阅读 → **系统自动记录「已读」和「完成」状态** → 回到年级列表时看到进度条 → 完成后解锁成就 → 有动力继续探索下一个年级。

### 2.2 分三层设计

| 层级 | 触发方式 | 存储 | 视觉反馈 |
|------|---------|------|---------|
| L1 板块浏览 | 折叠卡片展开 ≥ 5 秒 | `grade_read_{grade}` | 列表页卡片标题旁显示 👁 已读标记 |
| L2 清单完成 | 点击勾选（已有） | `{grade}_checklist` | 清单进度条（已有，需扩展） |
| L3 全板块完成 | 该年级所有板块 L1 全部达成 | `grade_mastered_{grade}` | 年级列表卡片变金色边框 + 🏅徽章 |

```javascript
// ============================================
//  卡片进度系统（v5.0 新增）
// ============================================

/** 获取进度数据 */
_getGradeProgress() {
  try {
    return JSON.parse(localStorage.getItem('grade_progress') || '{}');
  } catch (e) { return {}; }
},

/** 保存进度数据 */
_saveGradeProgress(progress) {
  localStorage.setItem('grade_progress', JSON.stringify(progress));
},

/** 初始化某年级进度（首次进入时调用） */
_initGradeProgress(grade) {
  const card = this.GRADE_CARDS.find(c => c.grade === grade);
  if (!card) return;
  const progress = this._getGradeProgress();
  if (!progress[grade]) {
    progress[grade] = {
      items: card.items.map((item, idx) => `${grade}-item-${idx}`),
      mastered: false
    };
    this._saveGradeProgress(progress);
  }
},

/** 标记板块为已读（L1） */
_markItemRead(grade, itemIndex) {
  const progress = this._getGradeProgress();
  if (!progress[grade]) return;
  const itemKey = `${grade}-item-${itemIndex}`;
  if (!progress[grade].readItems) progress[grade].readItems = [];
  if (!progress[grade].readItems.includes(itemKey)) {
    progress[grade].readItems.push(itemKey);
    this._saveGradeProgress(progress);
    this._checkGradeMastered(grade);
    this._updateGradeCardProgress(grade);
  }
},

/** 检查某年级是否全部阅读完成（L3） */
_checkGradeMastered(grade) {
  const card = this.GRADE_CARDS.find(c => c.grade === grade);
  if (!card) return;
  const progress = this._getGradeProgress();
  if (!progress[grade]) return;
  const totalItems = card.items.length;
  const readCount = (progress[grade].readItems || []).length;
  if (readCount >= totalItems && !progress[grade].mastered) {
    progress[grade].mastered = true;
    this._saveGradeProgress(progress);
    // 触发成就提示
    this._showMasteryBadge(grade);
    // 联动活动系统：解锁"年级达人"成就
    if (typeof ActivitySystem !== 'undefined') {
      ActivitySystem.unlockAchievement(`grade_${grade}_mastered`);
    }
  }
},

/** 展示达成徽章动画 */
_showMasteryBadge(grade) {
  const names = {
    freshman: '🌱 好奇宝宝',
    sophomore: '🌿 方向舵手',
    junior: '🌳 准鹅厂人',
    senior: '🚀 求职勇士'
  };
  const name = names[grade] || '年级达人';
  // 复用现有的 toast / 通知机制
  if (typeof UI !== 'undefined' && UI.showToast) {
    UI.showToast(`🏅 解锁成就：${name}！`, 3000);
  }
  // 企鹅做一个庆祝动作
  if (typeof PenguinEngine !== 'undefined') {
    PenguinEngine.playEmote('celebrate');
  }
},

/** 更新年级列表卡片上的进度条 */
_updateGradeCardProgress(grade) {
  const card = this.GRADE_CARDS.find(c => c.grade === grade);
  if (!card) return;
  const progress = this._getGradeProgress();
  const pg = progress[grade];
  const total = card.items.length;
  const read = pg ? (pg.readItems || []).length : 0;
  const pct = total > 0 ? Math.round((read / total) * 100) : 0;

  // 更新 DOM 中的进度条（如果在年级列表页）
  const gradeCard = document.querySelector(`.grade-page-card[data-grade="${grade}"]`);
  if (!gradeCard) return;

  // 移除旧进度条
  const oldBar = gradeCard.querySelector('.gpc-progress');
  if (oldBar) oldBar.remove();

  // 添加新进度条
  if (pct > 0 && pct < 100) {
    const bar = document.createElement('div');
    bar.className = 'gpc-progress';
    bar.innerHTML = `
      <div class="gpc-progress-track">
        <div class="gpc-progress-fill" style="width:${pct}%"></div>
      </div>
      <span class="gpc-progress-text">${read}/${total}</span>`;
    gradeCard.appendChild(bar);
  }

  // L3 完成：金色边框
  if (pg && pg.mastered) {
    gradeCard.classList.add('mastered');
    if (!gradeCard.querySelector('.gpc-master-badge')) {
      const badge = document.createElement('span');
      badge.className = 'gpc-master-badge';
      badge.textContent = '🏅';
      gradeCard.querySelector('.gpc-top').appendChild(badge);
    }
  }
},

/** 监听折叠卡片展开事件（在 toggleCollapse 中调用） */
_onCollapseOpen(grade, itemIndex) {
  // 设置 5 秒计时器，超时后标记为已读
  const timerKey = `read_timer_${grade}_${itemIndex}`;
  if (this[timerKey]) clearTimeout(this[timerKey]);
  this[timerKey] = setTimeout(() => {
    this._markItemRead(grade, itemIndex);
    delete this[timerKey];
  }, 5000);
}
```

## 设计决策记录

| 决策 | 结论 | 理由 |
|------|------|------|
| 进度不依赖服务器 | 全部 localStorage | 产品当前无后端，保持零依赖 |
| 兴趣卡片不阻断浏览 | 内嵌组件，非弹窗 | 不打断用户体验，"温柔的引导" |
| CSS 命名规范 | 大一 `fr-` / 大二新增 `so-` / 大三新增 `jr-` / 大四 `sr-` | 与现有代码一致 |
| 不引入第三方分析 SDK | 先不做 | MVP 阶段保持轻量 |
| L3 达成通知 | 复用现有 toast + 企鹅动画 | 避免重复造轮子 |
