## 三、页面流转图

```
首次访问 → P1领蛋页 → 点击领取 → 蛋动画 → P2年级选择 → P3主界面
                                                                       ↓
再次访问 ────────────────────────────────────────────────────────→ P3主界面（首页视图）
                                                                       ↓
                                    ┌── 💬 聊天 → 聊天覆盖层（从底部滑入）
                                    │       └── 返回首页 → tab高亮清除，回到默认首页
                                    ├── 📓 日记 → 日记子页面 → 返回首页
                                    ├── 📋 简历 → 简历模板页 → 返回首页
                                    ├── 📚 知识库 → 知识库子页面 → 返回首页
                                    ├── 🎓 年级直达 → 年级切换弹窗
                                    └── 📊 成长档案 → 志愿/活动页面 → 返回首页
```

> **Tab 高亮规则**：从子页面返回首页后，所有 tab 高亮自动清除，保持样式统一。进入子页面时对应 tab 高亮。
