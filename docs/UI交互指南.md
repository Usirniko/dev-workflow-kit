## 三、动效规范

### 过渡动画

| 动画 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 页面切换 | 300ms | ease-in-out | opacity + translateY |
| 弹窗出现 | 250ms | cubic-bezier(0.34,1.56,0.64,1) | scale + opacity |
| 聊天覆盖层滑入/出 | 380ms | cubic-bezier(0.4,0,0.2,1) | translateY（从底部） |
| 子页面滑入 | 350ms | cubic-bezier(0.4,0,0.2,1) | translateX（从右侧） |
| 蛋掉落 | 600ms | cubic-bezier(0.22,0.61,0.36,1) | translate + rotate |
| 选项卡切换 | 200ms | ease | 指示器滑动 |
| 消息出现 | 300ms | ease-out | translateY + opacity |
| 进化动画 | 1000ms | ease-in-out | 粒子+缩放+旋转 |
| 按钮hover | 150ms | ease | scale(1.02) + shadow |
| 维度条增长 | 500ms | ease-out | width动画 |
| 问候语轮换 | 每10秒 | — | 随机选一条（按时段匹配） |
| 时钟刷新 | 每秒 | — | HH:MM:SS 实时更新 |

### 企鹅动画

| 动画 | 说明 |
|------|------|
| 呼吸动画 | 持续，轻微缩放(0.98-1.02)，通过帧跳过机制减半速度，约4s周期 |
| 眨眼动画 | 随机触发，每3-8秒一次（blinkTimer 不受帧跳过影响） |
| 翅膀摆动 | 随呼吸同步，通过 frameCount 控制摆动幅度 |
| 配饰旋转 | 如思维齿轮等配饰旋转速度减半，更柔和 |
| 粒子特效 | 粒子的产生和消散速度随帧跳过减半 |

> **动画速度控制**：企鹅动画使用 `requestAnimationFrame` 驱动，通过 `frameSkip`/`frameSpeed` 机制（当前 `frameSpeed=2`）每2帧才推进一次 `frameCount`，所有依赖 frameCount 的动画速度减半。三个独立渲染器（主页企鹅、导航栏 mini 企鹅、"我的鹅"大图）均采用一致的速度控制。

### 性能要求

- Canvas渲染帧率 ≥ 30fps
- CSS动画使用 transform + opacity（GPU加速）
- 避免频繁DOM操作
- 使用 requestAnimationFrame 驱动Canvas

## 四、移动端适配

### 断点设计

```css
/* 手机竖屏 */
@media (max-width: 480px) { }

/* 手机横屏 / 小平板 */
@media (max-width: 768px) { }

/* 平板 */
@media (max-width: 1024px) { }
```

### 移动端特殊处理

- 输入框在移动端固定底部，避免键盘遮挡
- 按钮最小触摸区域 44×44px
- Canvas根据屏幕宽度动态缩放
- 年级选择弹窗在移动端占满屏幕
- 聊天消息气泡最大宽度80%
