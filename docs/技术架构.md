## 二、AI 对话对接逻辑

### DeepSeek API（需用户自行配置 API Key）

本项目使用 **DeepSeek API** 驱动 AI 对话，**用户需要在设置中配置自己的 API Key**（在 DeepSeek 开放平台获取）。

```javascript
// DeepSeek API 配置
const AI_CONFIG = {
  endpoint: 'https://api.deepseek.com/v1/chat/completions',
  model: 'deepseek-chat',
  temperature: 0.8,
  max_tokens: 512,
  timeout: 15000,
  maxRetries: 2,
  retryDelay: 1000
};
```

### 三级匹配策略

```
用户输入消息
    ↓
1. 离线问答库匹配（OfflineQA.match）
    ├─ 命中 (score >= 0.55) → 直接返回，无需 API
    └─ 未命中 ↓
          ↓
2. 构建消息数组（系统提示词 + 对话历史 + 用户消息）
    ↓
3. DeepSeek API 调用（fetch + Authorization Header）
    ├─ 成功 → 解析 JSON → 提取回复内容 → 返回
    └─ 失败 → 重试（最多2次，指数退避）
         └─ 全部失败 → 返回友好兜底回复
```

### DeepSeek API 调用示例

```javascript
async function sendMessage(userMessage, contextWindow, systemPrompt) {
  const apiKey = ChatEngine.getApiKey();
  if (!apiKey) {
    throw new Error('请先在设置中配置 DeepSeek API Key');
  }

  const messages = [
    { role: 'system', content: systemPrompt },
    ...contextWindow.slice(-10).map(m => ({ role: m.role, content: m.content })),
    { role: 'user', content: userMessage }
  ];

  const response = await fetch('https://api.deepseek.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`
    },
    body: JSON.stringify({ model: 'deepseek-chat', messages, temperature: 0.8, max_tokens: 512 })
  });

  const data = await response.json();
  return data.choices[0].message.content;
}
```

### 异常处理策略

| 异常类型 | 处理方式 | 用户提示 |
|----------|----------|----------|
| API Key 未配置 | 不请求 | "请先在设置中配置 DeepSeek API Key" |
| 网络错误 | 重试2次 | "网络连接失败，正在重试..." |
| 超时(15s) | 重试1次 | "请求超时，请检查网络后重试" |
| 401 未授权 | 不重试 | "API Key 无效，请检查设置" |
| 429 限流 | 等待后重试 | "请求太频繁，请稍后再试" |
| 5xx 服务端 | 重试2次 | "AI 服务暂时不可用，请稍后重试" |
| JSON 解析失败 | 不重试 | "响应格式异常，请重试" |

### 特征更新算法

```javascript
function updateAttributes(message, penguinData) {
  const keywords = extractKeywords(message);
  const changes = { knowledge: 0, creativity: 0, social: 0, action: 0, mentality: 0 };

  // 遍历关键词，累加各维度变化
  for (const keyword of keywords) {
    for (const [dimension, rules] of Object.entries(KEYWORD_MAP)) {
      for (const rule of rules) {
        if (matchKeyword(keyword, rule.pattern, rule.matchType)) {
          changes[dimension] += rule.score;
          break;
        }
      }
    }
  }

  // v3.1+ 消息质量权重（三因子乘积 0.4~3.0）
  const qualityMultiplier = calculateMessageQuality(message); // 长度×因果链×反思词

  // 应用连续登录倍率
  const consecutiveBonus = getConsecutiveBonus(penguinData.consecutiveDays);

  // 最终倍率 = 质量权重 × 连续登录倍率
  const finalMultiplier = qualityMultiplier * consecutiveBonus;

  // 更新属性（上限100）
  for (const dim of Object.keys(changes)) {
    penguinData.attributes[dim] = Math.min(
      100, 
      penguinData.attributes[dim] + Math.round(changes[dim] * finalMultiplier)
    );
  }

  // 更新经验值 = 五维属性总和
  penguinData.exp = Object.values(penguinData.attributes).reduce((a, b) => a + b, 0);

  return { changes, penguinData };
}
```

## 四、localStorage存储设计

### 存储键名

```javascript
const STORAGE_KEYS = {
  PENGUIN: 'penguin_data',
  CHAT: 'chat_history',
  CONFIG: 'user_config',
  STATE: 'app_state',
  VERSION: 'data_version'
};
```

### 安全读写封装

```javascript
const Storage = {
  // 安全写入
  set(key, data) {
    try {
      const serialized = JSON.stringify(data);
      localStorage.setItem(key, serialized);
      return true;
    } catch (e) {
      if (e.name === 'QuotaExceededError') {
        // 存储空间不足，清理旧聊天记录
        this.cleanupOldChats();
        try {
          localStorage.setItem(key, JSON.stringify(data));
          return true;
        } catch (e2) {
          console.error('Storage full, cleanup failed');
          return false;
        }
      }
      console.error('Storage write error:', e);
      return false;
    }
  },

  // 安全读取
  get(key, defaultValue = null) {
    try {
      const raw = localStorage.getItem(key);
      if (raw === null) return defaultValue;
      return JSON.parse(raw);
    } catch (e) {
      console.error('Storage read error:', e);
      // 尝试修复损坏数据
      localStorage.removeItem(key);
      return defaultValue;
    }
  },

  // 清理旧聊天记录
  cleanupOldChats() {
    const chat = this.get(STORAGE_KEYS.CHAT, { messages: [] });
    if (chat.messages.length > 50) {
      chat.messages = chat.messages.slice(-30); // 保留最近30条
      this.set(STORAGE_KEYS.CHAT, chat);
    }
  },

  // 数据导出
  export() {
    const data = {};
    for (const [name, key] of Object.entries(STORAGE_KEYS)) {
      data[name] = this.get(key);
    }
    return JSON.stringify(data, null, 2);
  },

  // 数据导入
  import(jsonStr) {
    try {
      const data = JSON.parse(jsonStr);
      for (const [name, key] of Object.entries(STORAGE_KEYS)) {
        if (data[name]) {
          this.set(key, data[name]);
        }
      }
      return true;
    } catch (e) {
      return false;
    }
  }
};
```
