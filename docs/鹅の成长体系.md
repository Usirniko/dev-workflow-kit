## 二、特征维度定义

企鹅拥有5个核心特征维度，每个维度0-100分，通过与AI的对话内容动态变化。

### 维度一览

| 维度 | 图标 | 说明 | 影响外观 |
|------|------|------|----------|
| **知识力** 📚 | 📚 | 学术讨论、专业知识 | 眼镜大小/书本配饰 |
| **创造力** 🎨 | 🎨 | 创意想法、艺术话题 | 颜色饱和度/创意光环 |
| **社交力** 💬 | 💬 | 人际交往、沟通话题 | 表情丰富度/围巾 |
| **行动力** ⚡ | ⚡ | 计划执行、目标达成 | 翅膀大小/动态效果 |
| **心态值** 💪 | 💪 | 积极心态、情绪管理 | 整体姿态/光环颜色 |

### 特征更新规则

```
每次对话结束后：
  1. 分析用户消息关键词
  2. 匹配对应维度（见技术架构.md关键词映射表）
  3. 该维度 +2~5 分（随机）
  4. 所有维度总和影响经验值增长
  5. 经验值 = SUM(各维度分数)
```

### 经验值计算

```
经验值 = 知识力 + 创造力 + 社交力 + 行动力 + 心态值
```
